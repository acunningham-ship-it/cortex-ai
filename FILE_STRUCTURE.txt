INSTALLATION & USAGE
====================

Quick Start (3 steps):

1. Install dependencies:
   cd cortex-ai/backend
   pip install -r requirements.txt

2. Configure (optional - defaults work):
   cd cortex-ai
   cp config.example.yaml cortex.yaml

3. Run server:
   cd cortex-ai/backend
   python -m uvicorn main:app --reload --port 7337

Then visit:
  http://localhost:7337/health
  http://localhost:7337/docs (OpenAPI documentation)
  http://localhost:7337/api/models
