Features

Everything you need for local AI development

One tool that replaces a dozen — without the cloud dependency or the monthly bill.

💬

Multi-Model Chat

Chat with any Ollama model or cloud API from one beautiful interface. Switch models mid-conversation. Full streaming support with markdown rendering.

⛓️

Pipeline Builder

Chain multiple AI calls together: summarize → translate → critique. Build reusable workflows in YAML or via the visual UI. Run from CLI with a single command.

📋

Prompt Library

Save and reuse prompts with {{variable}} substitution. 6 built-in templates to start: code review, summarize, debug, translate, and more.

⚡

Powerful CLI

Quick queries, piped input, pipeline execution, model listing. Works great in shell scripts. Integrates with any UNIX workflow.

📊

Usage Dashboard

Track token usage, latency, and request history across all models. See how much you're saving vs. cloud API pricing.

🔌

Plugin System

Extend with custom Python or JavaScript plugins. Built-in: file reader, web search, calculator. Add your own in minutes.

🔑

Multi-Provider

Ollama (local, zero cost), Claude (Anthropic), and OpenAI — all with the same interface. Config-driven. Add API keys only if you want cloud models.

🛡️

Privacy First

All data stays on your machine. SQLite database, no telemetry, no accounts. Your conversations and prompts are yours — always.

🚀

Zero Config Demo

Run cortex serve. Browser opens. Ollama auto-detected. Ready to chat in 30 seconds. No signup, no API keys, no config files needed.

Installation

Up and running in 60 seconds

1

Install Cortex AI

pip install cortex-ai
2

Make sure Ollama is running (optional but recommended)

# Install Ollama from ollama.ai, then:
ollama pull qwen2.5:7b
3

Start Cortex AI

cortex serve
# Browser opens automatically at http://localhost:7337
4

Optional: Add cloud API keys (cortex.yaml)

# cortex.yaml (in your project directory)
ollama_url: "http://localhost:11434"
claude_api_key: "sk-ant-..." # optional
openai_api_key: "sk-..." # optional
How it works

Simple architecture, powerful results

Cortex runs a local FastAPI server that proxies requests to Ollama or cloud APIs, stores history in SQLite, and serves a React UI — all in one pip install.

01

Install & Run

One pip install. Cortex auto-detects Ollama and any configured API keys.

02

Pick a Model

Choose from local Ollama models or cloud APIs — all in the same dropdown.

03

Build Workflows

Create pipelines and templates. Run them from the UI, CLI, or REST API.

04

Track & Optimize

Dashboard shows usage, latency, and savings vs. cloud API alternatives.
